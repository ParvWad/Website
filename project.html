DiscordBot

What is Discord/DiscordBot?

Discord is a voice, video, and text chatting app that gives the user's the ability to communicate
with voice, video, text, media in groups called servers. Discord also has an API,
which allows developers to build 'bots' and allows programmers to interact with Discord.
This API was utilized to develop my own 'Discordbot' which allowed users to maintain an amazon shopping list!
